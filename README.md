# Martina · Cuentos para dormir

Sitio web de cuentos infantiles con temática de ajedrez, inspirados en el alter ego Martina: una niña ajedrecista valiente, agresiva, soñadora y teórica que no le gusta empatar.

Cada cuento es una aventura surrealista —al estilo de *Alicia en el país de las maravillas* y *Viva la Reina* (*Lang leve de koningin*)— que entrelaza conceptos reales de ajedrez con reinos mágicos y personajes inolvidables.

## Estructura

```
martina/
├── index.html          # Página de inicio con presentación de Martina
├── cuentos/
│   └── 01-el-primer-movimiento.html  # Primer cuento
├── css/
│   └── style.css       # Estilos mobile-first
├── js/
│   └── main.js         # Interactividad del sitio
└── README.md
```

## Cómo ver el sitio

Abre `index.html` en cualquier navegador. No requiere servidor ni dependencias.

O con un servidor local rápido:

```bash
python3 -m http.server 8000
# Abrir http://localhost:8000
```

## Personaje: Martina

- **Estilo de juego**: agresiva y caótica, base teórica sólida (comprende ajedrez posicional pero elige el ataque)
- **Filosofía**: no le gusta empatar; prefiere arriesgar por la victoria
- **Ídolos**: Judith Polgar (precisión) y Mikhail Tal (creatividad)
- **Personalidad**: valiente, justiciera, imaginativa, determinada, segura de sí misma

## Inspiración

Historias originales creadas por un papá para leerle a su hija antes de dormir. Cada cuento busca motivar para torneos de ajedrez mientras enseña conceptos reales del juego en un tono surrealista y poético.

# Código de Conducta

## Nuestro compromiso

Este es un proyecto para niños. Nos comprometemos a mantener un espacio acogedor, respetuoso y libre de acoso para todas las personas, sin importar edad, género, experiencia, nacionalidad o cualquier otra característica.

## Estándares de comportamiento

**Comportamiento esperado:**
- Empatía y amabilidad
- Respeto por opiniones diferentes
- Críticas constructivas y respetuosas
- Enfocarse en lo mejor para la comunidad
- Usar lenguaje apropiado para todas las edades

**Comportamiento inaceptable:**
- Lenguaje ofensivo, insultos o ataques personales
- Acoso de cualquier tipo
- Publicar información privada de otros sin permiso
- Cualquier conducta inapropiada para un proyecto infantil

## Aplicación

Los mantenedores del proyecto son responsables de hacer cumplir este código. Las violaciones pueden resultar en advertencias, eliminación de comentarios, o expulsión temporal/permanente.

## Reportes

Si experimentas o presencias una violación, repórtala a través de un issue privado o contactando al mantenedor.
